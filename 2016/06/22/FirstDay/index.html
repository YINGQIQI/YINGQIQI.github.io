Hexo 博客搭建步骤

安装nodeJS：

  • 进入nodeJS官网下载nodeJS (官网地址：https://nodejs.org/en)
  • 安装成功后在终端进行测试：输入node -v，回车查看，如下图所示则安装成功

安装hexo：

  • 在终端输入npm install hexo -g,然后回车
  • 若此时终端出现root Administrator等错误时，需重新输入sudo npm install hero -g，以管理员方式安装，回车。
  • 输入管理员密码，回车
  • 测试hexo是否安装成功：在终端输入hexo，回车查看，如在终端出现如下图所示的代码，则安装成功。

创建目录并进入(以桌面创建为例):

  • 在终端输入“cd Desktop”,回车,进入电脑桌面
  • 在终端输入“mkdir blog”，回车，在桌面创建一个名为blog的文件夹
  • 在终端输入“cd blog”,回车，进入名为blog的文件夹
  • 在终端输入“hexo init”，回车，初始化hexo的目录
  • 在终端输入“hexo server”回车，启动本地服务，在成功启动本地服务器后会出现一个本地博客地址：http://localhost:4000/,点击进入。(按住ctrl+c退出hexo运行)，如下图所示：
  • 在进入hexo博客的网站后注册一个账号并登陆，步骤如下图所示：

1、进入注册界面，点击右上角的“sign in”

2、填写用户名、注册邮箱和密码

注册github账号：

  • 首先需要配置ssh-key的值：在终端输入“ssh-keygen”，连续回车三次,如下图所示:

  • 在终端输入“vim ~/.ssh/id_rsa.pub”复制ssh-key的值。

  • 打开本地博客地址，点击用户名旁边的settings，点击personal settings 里面的SSH and GPG keys，在key的输入框内把之前复制的ssh-key的值输入进去，点击ADD SSH key确认。这样一个博客账号就创建好了。



  • 在终端输入“：q”，回车，进入博客文件。

  • 下面将配置github与本地机器验证：
    在终端输入“git config –global user.name”用户名”（例如：git config –global user.name”YINGQIQI”）
    在终端输入“git config –global user.email”邮箱地址”（例如：git config –global user.email”2544360251@qq.com”）
  • 新建博客域名仓库：


    [注意：]repositary的名字必须和用户名一致

配置config.yml：

打开桌面上的blog文件夹中的config.yml文件 在deploy字段的下面的type后面填入“ git”；进入自己本地博客地址把域名仓库的 https的域名地址复制到deploy字段下面的repo后面

发布：

  • 在终端输入“cd Desktop”，回车,然后输入“cd blog”，回车，进入blog文件夹中
  • 在终端输入：“npm install hexo-deployer-git –save”
  • 在终端输入：“hexo g”，回车，把一些文件转换成css和html文件
  • 在终端输入：“hexo d”，回车，把生成的网页发布到仓库。
  • 打开博客地址查看效果

博客使用：

进入博客

新建博客的网址为：https://**用户名**.github.io/

更换博客主题

  • 可以在网上下载一些hexo博客的主题，clone主题的https地址
  • 在终端进入blog里面的themes文件夹中，回车，然后输入“git clone 主题http地址”回车（此过程需要一段时间），例如下图所示：
  • 此时打开blog文件夹的themes中可以发现多出一个主题文件夹
  • 在blog文件夹的_config.yml文件中修改主题并保存

    更换博客名称

  • 在_config.yml文件中可以修改主题和博客题目以及作者语言等，在修改完后需保存并重新在终端运行

更新博客

  • 在终端输入“hexo new”,创建新日志
  • 在终端输入“hexo server”，在本地服务器查看博客内容是否有问题
  • 在终端输入“hexo g”把md文件转化成html文件和css文件
  • 在终端输入“hexo d”把本地写的博客发布到仓库
